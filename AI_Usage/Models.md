# AI Models Used

This document records the AI systems used throughout the development of this project.

| AI Tool | Primary Role | Used For |
|---------|--------------|-----------|
| NotebookLM | Research Assistant | Reading mentor-provided PDFs, summarizing literature, understanding methodology, organizing references |
| ChatGPT (GPT-5.5) | Development Assistant | Dataset construction, causal inference implementation, debugging, documentation, GitHub organization |
| GitHub Copilot *(Future if used)* | Programming Assistant | Code completion (not currently used) |

---

# NotebookLM

NotebookLM was used from the beginning of the internship as instructed by the project mentor.

Its primary objective was to facilitate understanding of the research material supplied during the internship.

NotebookLM assisted in:

- summarizing research papers and the book on Causal Reasoning,
- understanding mentor-provided documentation,
- explaining causal inference terminology,
- identifying important methodological concepts,
- organizing literature,
- locating relevant information within lengthy documents.

NotebookLM was **not** used to generate the project's implementation or statistical conclusions.

---

# ChatGPT (GPT-5.5)

ChatGPT served as a development assistant throughout the project.

Major areas of assistance included:

- designing the synthetic benchmark dataset,
- planning the real-world validation dataset,
- preprocessing multiple public datasets,
- causal inference implementation,
- debugging Python code,
- GitHub repository organization,
- Streamlit application planning,
- LaTeX report preparation,
- documentation writing.

All generated code and documentation were manually reviewed and modified before inclusion.

---

# Human Responsibilities

The project author remained responsible for:

- research design,
- causal assumptions,
- variable selection,
- statistical validation,
- implementation verification,
- interpretation of results,
- scientific conclusions.

AI functioned only as an assistive tool.
