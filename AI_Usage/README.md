# AI Usage Documentation

## Purpose

This folder documents the use of Artificial Intelligence (AI) tools during the development of this project.

The objective of this documentation is to maintain transparency, reproducibility, and academic integrity regarding AI-assisted research and software development.

The project followed a **human-in-the-loop** workflow. AI tools were used to improve productivity, understand research material, assist with implementation, generate documentation drafts, and organize the project. However, all statistical reasoning, methodological decisions, implementation choices, validation procedures, and interpretation of results were independently reviewed and finalized by the project author.

---

## AI Tools Used

| Tool | Primary Purpose |

| NotebookLM | Literature review, mentor-provided document analysis, research understanding |
| ChatGPT (GPT-5.5) | Programming assistance, causal inference implementation, debugging, documentation, GitHub organization |
| GitHub Copilot *(Future if used)* | Code completion |

## AI Usage Policy

AI was used strictly as an assistive technology.

AI did **not** replace:

- statistical reasoning,
- causal assumptions,
- scientific interpretation,
- implementation validation,
- or final research conclusions.

All outputs generated by AI were manually reviewed before incorporation into the project.
